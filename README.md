# Isaiah Thomas Portfolio

This is a Next.js project built with Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/page.tsx`: The main page component containing the logic for switching between projects and the About section.
- `app/layout.tsx`: The root layout ensuring full-screen height and importing fonts.
- `app/globals.css`: Global styles and Tailwind imports.

## Deploy on Vercel

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com/new).

1. Upload this folder to your GitHub.
2. Import the project on Vercel.
3. It should automatically detect the settings and deploy!
